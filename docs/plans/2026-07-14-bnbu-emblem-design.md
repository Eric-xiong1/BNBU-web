# BNBU 校徽首页落位设计

## 目标

在学生网页版首页复用 Android 端已有的 BNBU 官方校徽，并与用户提供的移动端参考图保持一致：校徽出现在首页问候区左侧，同时作为底部“首页”导航的识别图标。

## 已确认方案

- 资产来源：`BNBU-Sports-Android-reference-20260714/app/src/main/res/drawable/bnbu_emblem.xml`。保留原始 `83 × 83` viewBox、路径数据与官方品牌色 `#0166A4`，只做 Android Vector Drawable 到标准 SVG 的格式转换。
- 问候区：在姓名与学期说明左侧放置 56px 校徽容器。容器采用当前页面的 surface、细边框和小圆角，不改动校徽本体。
- 底部导航：“首页”使用校徽，其他四个标签保持现有 Material 图标。校徽始终可见；激活态沿用现有浅蓝色胶囊背景，非激活态降低透明度。
- 兜底：校徽图片下方保留现有 home SVG；如果本地 SVG 资源加载失败，仍能显示首页图标。
- 无障碍：首页问候区的校徽提供“BNBU 校徽”替代文本；底部图标视为装饰，导航按钮仍由“首页”文本提供名称。

## 验收标准

1. 首页问候区能看到清晰、不拉伸的蓝色 BNBU 校徽。
2. 底部“首页”标签显示同一校徽，激活态背景与其余导航交互保持一致。
3. 390×844、768×1024、1440×900 三种视口无水平溢出，深色模式下校徽容器仍有足够对比度。
4. 现有学生端单元测试、静态烟测和 Playwright 浏览器验收全部通过。
