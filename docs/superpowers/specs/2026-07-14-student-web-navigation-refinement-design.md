# 学生 Web 端导航精简与记录缩略图设计

## 目标

突出运动打卡这一核心动作，移除冗余首页，并让学生在记录列表中直接识别已上传凭证。

## 已确认设计

- 删除首页入口、首页路由和首页视图；登录后仍默认进入运动打卡。
- 底部导航分为上下两层：内容区底部中央放置蓝色“＋ 打卡”胶囊；其下由分隔线隔开，等宽排列“课程、成绩、我的”。
- 打卡胶囊完整位于内容区，不覆盖三项导航，并在所有登录后页面固定显示。内容区预留底部空间，最后一项可滚动到按钮上方。
- 成绩页保持现状。
- 免测申请仅作为“我的”中的二级入口，不在其他主页面重复出现。
- 打卡记录卡片显示第一份有效凭证的缩略图；多凭证显示 `+N`，视频显示播放标识，无凭证或加载失败时显示运动类型占位。
- 点击记录卡片进入详情；详情以真实图片或视频预览全部凭证，不再使用“查看凭证”文字占位。

## 验收标准

- 底部只出现课程、成绩、我的三个普通按钮，打卡胶囊水平居中且位于分隔线上方。
- 页面中不再出现首页导航或首页路由。
- 手机、平板、桌面均无页面级横向溢出，底部内容不被固定区域遮挡。
- 演示记录和真实 `/uploads/*` 记录均能显示缩略图或明确占位。
- 原有成绩、课程、通知、免测、上传和补交材料流程不回归。
